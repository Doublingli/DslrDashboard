DSLR Dashboard
DSLR Dashboard is an Android application for controlling your DSLR camera (primary for Nikon)  with USB. Unfortunatly not all Android phones include the required USB support. Currently some of the Android 2.3.x and 3.x have the required USB support and Android 4.0.x should have it on all devices.
You can find the application on Google Play for free: https://play.google.com/store/apps/details?id=com.dslr.dashboard&feature=search_result#?t=W251bGwsMSwxLDEsImNvbS5kc2xyLmRhc2hib2FyZCJd
To use it you need a cheap USB OTG adapter or cable that you can find on internet.(http://www.dealextreme.com/p/micro-usb-on-the-go-host-otg-adapter-pair-50774)
Some of the application features:
  • reading/writing camera properties
  • initiating capture (to your camera and phone)
  • entering live view (manual focus, zoom in/out, display face fcous)
  • custom bracketing up to 11 images
  • timelapse
  • image preview (RAW NEF format to)
  • sound/noise initiated capture
There is an open discussion on xda-forums: http://forum.xda-developers.com/showthread.php?t=1256583
Feel free to comment, suggest or report errors there.

DSLR Dashboard main view


Notice: On tablets or phones the display is a bit different if the width of the screen is equal or higher then 1280, then the bracketing view is combined into the main view.
  1. Overlay button that is used to display or hide the buttons from 6 to 30 when in live view mode
  2. Initiate capture. The image will be saved to camera sdcard. If focus is not MF (manual focus) and if your lens can support focusing the camera will always perform autofocus before initiating the capture. If autofocus fails the capture will be not initiated.
  3. Initiate capture. The image will be transfered to your phone. All from above apply for this to, only the target media is different.
  4. Initiate autofocus if your lens can do it.
  5. DSLR Dashboard mode overlay button, will be described below.
  6. Camera shooting mode selector. Only can be changed if the camera is in host mode. See 8.
  7. Scene selector if Camera is in Scene mode (6)
  8. Camera mode selector. Initialy camera is in camera mode, meaning that your can operate controls on your camera (change shooting mode, initiate capture, change properties). If you put your camera in host mode then the application will have full control over your camera properties and the controls on camera will be disabled. You will be able to change the camera shooting mode (6).
  9. Start the live view mode. If for some reason the live view can't be activated you will get a dialog with the prohibition reason(s). (recording media (28) should be sdram to be able to enter live view)
  10. Camera battery level indicator
  11. Current aperture value. Depending camera shooting mode (6) you can change the aperture value
  12. Current exposure indicator in EV
  13. Current shutter speed. Depending camera shooting mode (6) you can change the shutter speed.
  14. Captured image size. Depending on camera shooting mode (6) and image quality seetings (19) you can change the captured image size.
  15. Current white balance settings, press to change
  16. Current focus metering mode, press to change
  17. Current focus mode, press to change. (MF selection is if you select the manual focus and MF fixed is when manual focus is selected on lens or the lens dosn't support focusing)
  18. Shooting operation mode (single, continuous , etc)
  19. Captured image quality current setting, press to change
  20. Current ISO setting, press to change
  21. Current exposure metering mode, press to change
  22. Current live view focus mode, press to change
  23. Number of continuous shooting frames, press to change (only have meaning if  shooting operation mode (18) is in continous shooting
  24. Current selected internal flash mode, press to change
  25. Internal flash exposure compensation, press to change
  26. Exposure compensation, press to change
  27. Exposure EV step (1/3 or 1/2)
  28. Recording media. SDCARD - captured image is saved to sdcard located in your camera, SDRAM - captured image is saved to camera internal sdram and transfered to your phone
  29. Current selected picture control  (standard, neutral, vivid, etc) press to change
  30. Current selected Active D-Ligthing, press to change
